# Jigsaw 1.6 - Strategic Planning Visualization

Interactive strategic planning visualization system with Logic Model and Matrix views.

## Features

- Logic Model visualization (Purpose - Outcomes - Value Chain - Resources)
- Contribution Map (Outcomes x Value Chain matrix)
- Development Pathways (Resources x Value Chain matrix)
- Convergence Map (Value Chain x External Factors matrix)
- Agents Canvas (Interactive flow visualization)
- Dark/Light theme support
- Multiple demo systems included

## Quick Start

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 16 + React 19
- Tailwind CSS + shadcn/ui
- TypeScript
- pnpm package manager

## Deployment

This project is configured for Vercel deployment with pnpm.
